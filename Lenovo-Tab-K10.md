# Lenovo Tab K10 (Lenovo TB-X6C6F)

Android 11 firmware is the only firmware that seems to work currently. Make sure to disable dm-verity, and use fastbootd to flash the image.

| Camera                    | Works                                                     |
| Speaker / Mic             | Works                                                     |
| Bluetooth                 | Works                                                     |
| SDCard                    | Works                                                     |
| MTP                       | Works                                                     |
| WiFi                      | Works                                                     |
| SIM / Mobile Data / Voice | None                                                      |
| VoLTE                     | None                                                      |
| Fingerprint               | None                                                      |
| NFC                       | None                                                      |
