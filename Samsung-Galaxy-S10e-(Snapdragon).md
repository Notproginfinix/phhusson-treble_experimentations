# Samsung Galaxy S10e 



## Steps to install

* No twrp:  Follow [this guide](https://forum.xda-developers.com/galaxy-s10/how-to/guide-how-to-install-custom-rom-using-t4114435) made by user kkoo on xda.
* TWRP: Flash gsi img file as system in twrp.
