# [Infinix Note 5] - [x604]

## Hardware Support

* Camera:
> OK.

* Speaker / Microphone
> OK.

* Bluetooth
> OK.

* Wifi
> OK.

* SIM / Mobile Data / Voice
> NOT OK (Mediatek uses a different way of handling RIL compared to Qualcomm).
> With some build.prop tweaks, it should be good! But we can't do that as of now (no root yet).

* VoLTE
> Not tested.

* Fingerprint Reader
> OK.
